----> LPs e termos
-> Pesquise um pouco sobre Apache, MySQL e PHP, pois vamos usar na próxima aula para construir nossa primeira aplicação-teste.
Apache - Programa de computador denominado servidor web. Software que processa as solicitações via HTTP
MySQL - banco de dados, que armazena todas as informações processadas em um formato inteligível
PHP - A linguagem de programação que cria conteúdos dinâmicos, além do código ser executado no servidor, impossibilitando o usuário de saber qual era o código fonte
